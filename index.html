Mobile Deployment Guide

// Android · Termux · Ubuntu · ARM64

An ultra-lightweight, open-source AI agent framework optimized for resource-constrained hardware — including RISC-V boards and Android devices. Built natively in Go. A bare-metal alternative to heavyweight frameworks like OpenClaw.

go / native arm64 proot-distro telegram bot ollama v0.2.2
Prerequisites
📱

Before starting, ensure Termux is installed on your Android device. All commands below are run inside Termux and the Ubuntu environment it provisions.

Installation Steps
01

Update Packages & Install Ubuntu

Bootstrap your Termux environment with a full Ubuntu container via proot-distro.

bash — termux
pkg update && pkg upgrade -y
pkg install proot-distro -y
proot-distro install ubuntu
proot-distro login ubuntu
02

Prepare the Ubuntu Environment

Once inside the Ubuntu shell (you'll see root@localhost), install the essential toolchain.

bash — ubuntu root
apt update && apt upgrade -y
apt install wget tar nano ca-certificates curl -y
03

Download & Setup PicoClaw

Fetch the ARM64 release binary, extract it, and move it into your system path.

bash — ubuntu root
wget https://github.com/sipeed/picoclaw/releases/download/v0.2.2/picoclaw_Linux_arm64.tar.gz

tar -xzvf picoclaw_Linux_arm64.tar.gz
chmod +x picoclaw
mv picoclaw /usr/local/bin/
04

Create the Config Directory

Set up the PicoClaw configuration directory and open the config file in the editor.

bash — ubuntu root
mkdir -p ~/.picoclaw
nano ~/.picoclaw/config.json
05

Paste & Configure config.json

Copy the template below. Replace the placeholder values with your Telegram Bot Token and User ID. Swap any Ollama-compatible model in the model field.

json — ~/.picoclaw/config.json
{
  "agents": {
    "defaults": {
      "workspace": "/root/.picoclaw/workspace",
      "model": "my-assistant"
    }
  },
  "model_list": [
    {
      "model_name": "my-assistant",
      "model": "ollama/kimi-k2.5:cloud",
      "api_base": "http://127.0.0.1:11434/v1",
      "api_key": "ollama"
    }
  ],
  "channels": {
    "telegram": {
      "enabled": true,
      "token": "YOUR_TELEGRAM_BOT_TOKEN",
      "allow_from": [YOUR_TELEGRAM_USER_ID]
    }
  }
}
⚠️ Replace YOUR_TELEGRAM_BOT_TOKEN with your token from @BotFather, and YOUR_TELEGRAM_USER_ID with your numeric Telegram ID.
06

Launch PicoClaw

Start the gateway process to bring your Telegram bot online.

bash — ubuntu root
picoclaw gateway
